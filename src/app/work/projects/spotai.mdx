# SpotAI: Exploring Musical Heritage Through AI

## Overview

SpotAI is a personal project born from curiosity about musical heritage and a desire to become more culturally aware through music. By combining Spotify's rich music data with OpenAI's analytical capabilities, this application provides users with deep insights into their listening habits and connects them with the historical and cultural significance of their favorite tracks. The project serves as both a learning tool and a way to discover the stories behind the music we love.

## Key Features

### Spotify Integration
Seamlessly connects with users' Spotify accounts to analyze listening history, favorite tracks, and musical preferences using the Spotify Web API.

### AI-Powered Analysis
Leverages OpenAI to generate detailed insights about artists, genres, and musical movements, providing cultural and historical context for users' music choices.

### Interactive Visualization
Creates engaging visualizations of listening patterns and musical connections using modern web technologies.

### Historical Context
Provides rich historical background about different musical eras, genres, and artistic influences that shape users' music preferences.
