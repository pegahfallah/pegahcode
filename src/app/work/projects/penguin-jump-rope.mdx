# Jump Rope Penguin Game

## Overview

A charming browser-based game featuring a penguin that must time its jumps to avoid an incoming jump rope. Simple controls and increasing difficulty make for an addictive gameplay experience.

## Features

- Simple one-button gameplay
- Increasing difficulty as the game progresses
- Cute penguin character animations
