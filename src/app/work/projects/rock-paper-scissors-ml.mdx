# Rock Paper Scissors ML

## Overview

A fun web application that uses Google Teachable Machine to recognize hand gestures and play rock paper scissors against the computer. The AI model was trained to identify rock, paper, and scissors hand positions in real-time through the webcam.

## Features

- Real-time hand gesture recognition
- AI-trained model using Google Teachable Machine
- Interactive gameplay against computer opponent
